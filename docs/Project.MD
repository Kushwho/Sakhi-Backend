We are building Sakhi, a voice-first AI companion app for Indian children aged 4–12, targeting urban nuclear family parents in India. The core product is an animated, interactive avatar character (think Talking Tom but with genuine AI intelligence) that children talk to naturally using their voice. Under the hood, Sakhi runs on LiveKit for real-time WebRTC audio transport, with a STT→LLM→TTS voice pipeline using Deepgram Nova-3 for multilingual speech-to-text (supporting Hindi, Tamil, Telugu, Kannada, Marathi, Bengali and English), GPT-4o-mini as the LLM brain, and ElevenLabs for expressive child-friendly TTS voice output. Silero VAD and LiveKit's MultilingualModel handle turn detection (critical because children pause mid-thought frequently). The agent is a Python LiveKit AgentSession that joins a LiveKit Room when a child opens the app — it carries the child's name, age, language, and profile as room metadata, and its system prompt defines Sakhi's personality: warm, curious, encouraging, age-appropriate, and strictly child-safe with no direct homework answers (Socratic guidance only). The agent exposes three key function tools: explain_concept (hits a CBSE/ICSE curriculum RAG system), log_emotion (writes detected emotional states to a Postgres DB that feeds the parent dashboard), and set_avatar_expression (publishes a LiveKit Data Channel message to the React frontend to trigger avatar BlendShape animations in Three.js — this is how Sakhi's face reacts in real time to what it's saying and feeling). The frontend is React with the LiveKit JS SDK handling audio and data channel events, and a Three.js avatar with a full expression system (happy, thinking, excited, concerned, sad, celebrating) driven by the agent's data messages. Alongside the child-facing product, there is a parent dashboard built in React that reads from the same Postgres DB and displays weekly emotion trend charts, subject mastery progress bars, social confidence scores from group sessions, and monthly milestone reports — this dashboard is the primary purchase justification for parents. The social layer allows up to 4 real-world friends to each have their own Sakhi companion, with the companions able to join shared LiveKit Rooms for group homework sessions, collaborative storytelling, and study quizzes — with a separate parent visibility layer showing group participation health. The business model is a freemium SaaS subscription (₹799–₹1,499/month) targeting urban Indian parents who are anxious about AI readiness and lonely children in nuclear families. The stack is: React frontend, Python FastAPI backend, LiveKit Cloud for WebRTC, Postgres for all persistent data, and the LiveKit Agents SDK for the voice pipeline. Build everything production-grade, child-safe by default, and with the parent dashboard and avatar expression bridge treated as first-class features — not afterthoughts.
